# Limitações desta análise

Este repositório apresenta uma análise conceitual baseada em padrões de engenharia de agentes.

Algumas limitações devem ser consideradas:

- Não há análise do código interno de projetos de terceiros.
- As conclusões se baseiam em comportamento observável.
- O estudo é arquitetural, não experimental.

O objetivo não é avaliar pessoas ou projetos específicos, mas demonstrar como diferentes decisões de arquitetura produzem comportamentos distintos em agentes de IA.
